Documentation Officielle - PMT

Logiciel de gestion de projet

1. Présentation du Projet

PMT est une plateforme de gestion de projet collaboratif conçue spécifiquement pour les équipes de développement logiciel. L'outil vise à centraliser la planification, le suivi et la collaboration.

Objectifs Clés :

  • Planification structurée des projets.
  • Suivi précis de l'avancement des tâches.
  • Collaboration fluide (Admin, Membre, Observateur).

2. Spécifications Fonctionnelles

Authentification & Accès

  • Inscription : Création de compte (Nom, Email, MDP).
  • Connexion : Accès sécurisé via Email/MDP.

Gestion de Projet & Équipe

  • Création : Un utilisateur crée un projet et devient Admin.
  • Invitation : Invitation de membres via leur adresse e-mail.
  • Rôles : Attribution des permissions (Admin, Membre, Observateur).

Gestion des Tâches

  • Opérations : Création, Assignation et Mise à jour des tâches.
  • Visualisation : Vue unitaire et Tableau de bord par statuts.
  • Audit : Historique complet des modifications et notifications e-mail.

3. Architecture Technique

Frontend

Angular 17+ (Signals, RxJS) & Tailwind CSS

Backend

Spring Boot 3+ (Java 17, JPA, Spring Security)

Database

MySQL

Auth

API REST sécurisée par JWT

4. Design System

Couleurs : Deep Navy (#1E293B) & Soft Gray (#F8FAFC)

Composants : Cartes (12px), Sidebar persistante.

Statuts : Success Warning Danger

Branches

  • fix/nom-du-bug/plateforme
  • feat/nom-fonctionnalite/plateforme

Workflow

develop_v1 (dev) ➔ staging_v1 (test) ➔ main_v1 (prod)

6. Roadmap

Phase 1 : Backend (Initialisation JPA, Sécurité JWT, CRUD).

Phase 2 : Frontend (Intégration Tailwind, Layouts, Services).

Phase 3 : Industrialisation (Tests, CI/CD, Déploiement).
