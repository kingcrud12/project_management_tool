Architecture du Serveur Spring Boot 3+

Structure des Dossiers (Maven)

src/main/java/com/pmt/api/
├── config/ // Security & Bean configs
├── controllers/ // REST Endpoints
├── dtos/ // Data Transfer Objects (MapStruct)
├── entities/ // JPA Models (Hibernate)
├── repositories/ // Database Access
└── services/ // Business Logic

Choix Techniques

1. Persistance & Données

Utilisation de Spring Data JPA avec MySQL. Les IDs sont gérés en INT AUTO_INCREMENT. Les relations sont définies en FetchType.LAZY pour optimiser les performances.

2. Sécurité Stateless

L'authentification est gérée par Spring Security. Un filtre personnalisé intercepte les requêtes pour valider le Authorization: Bearer contenu dans le header.
