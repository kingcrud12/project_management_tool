Référence de l'API REST v1

Base URL : http://localhost:8080/api/v1

POST /auth/login

Authentifie un utilisateur et retourne un Token JWT.

Request Body : 
{
  "email": "jean.dupont@email.com",
  "mdp": "password123"
}
GET /projects

Récupère la liste des projets auxquels l'utilisateur a accès.

Response : 200 OK 
[
  {
    "id": 1,
    "name": "E-commerce App",
    "role": "Admin"
  }
]
POST /tasks

Crée une nouvelle tâche (Réservé Admin/Membre).

Headers : Authorization: Bearer <token>
