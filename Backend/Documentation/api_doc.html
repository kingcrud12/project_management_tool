📚 Documentation API Projet

🔐

Authentification (/auth)

Méthode Route Description
POST /auth/register Inscription d'un nouvel utilisateur.
POST /auth/register/confirm Confirmation du compte (via code ou lien).
POST /auth/login Connexion et récupération du token.
POST /auth/logout Déconnexion et invalidation du token.
📁

Projets (/projects)

Méthode Route Description
POST /projects Créer un nouveau projet.
GET /projects Liste des projets (membre, admin ou observateur).
GET /projects/{id} Récupérer les détails d'un projet spécifique.
PATCH /projects/{id} Modifier les informations d'un projet.
POST /projects/{id}/members Inviter un nouveau membre au projet.
POST /projects/{id}/members/confirm Accepter l'invitation à rejoindre le projet.
✅

Tâches (/projects/{id}/tasks)

Méthode Route Description
POST /projects/{id}/tasks Créer une tâche dans un projet.
GET /projects/{id}/tasks/{taskId} Récupérer une tâche précise.
PATCH /projects/tasks/{taskId} Modifier le statut ou le contenu d'une tâche.
📜

Historique (.../history)

Méthode Route Description
POST /projects/{id}/tasks/{taskId}/history Enregistrer une nouvelle action/événement.
GET .../history/{historyId} Récupérer le détail d'une action spécifique.
